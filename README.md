### Local Development

1. **Start the Amplify Sandbox:**

   ```bash
   npm run sandbox
   ```

2. **Run the dev server** (in a new terminal):

   ```bash
   npm run dev
   ```

3. Visit http://localhost:5173 and click "Call API" to test!
